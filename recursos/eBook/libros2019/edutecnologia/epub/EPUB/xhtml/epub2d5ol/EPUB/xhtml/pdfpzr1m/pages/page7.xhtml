Según Palomares (2019) el concepto de clase se reere a las cosas que existen y nos rodean,
cosas u objetos de las que obtenemos sus características y funciones básicas que las denen
como dicho objeto, a esto se le llama abstracción, y posteriormente las pasamos a un ámbito
computacional. Podemos denir clase como una categoría de algo, por lo cual tiene atributos
(características) y métodos (funciones). De este modo, un diagrama de clases representa
la estructura y el comportamiento de cada uno de los objetos (instancias de las clases) del
sistema y sus relaciones con los demás objeto
El módulo Controlador contiene el Faces Servlet, pues el backend del sistema estará imple-
mentando con Java Server Faces. Y por último el módulo Vista, contendrá todas las interfaces
grácas del usuario.
Según Pérez (2007) se dene una base de datos como una serie de datos organizados y
relacionados entre sí, los cuales son recolectados y explotados por los sistemas de información
de una empresa o negocio en particular.
De este modo, se tiene que para el desarrollo de este sistema se emplearán dos tipos de
base de datos, la primera será base de datos relacional (implementada con MySQL) y la se-
gunda base de datos semi estructurada (implementada con XML).

TipoEjercicio, Pregunta y TipoPregunta. Y tiene otras 8 entidades que se derivan de las rela-
